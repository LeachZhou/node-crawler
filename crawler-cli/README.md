# 🐢 juejin-following

## 介绍

- juejin-following 是一个批量关注掘金用户爬虫，使用 Node.js 开发。

## 开发环境

- Node.js@8.11.1

## 安装

```
$ cd juejin-following
$ npm i
$ npm start
```

## 遇到问题

- 如果遇到 IP 限制问题，可以使用代理 IP 发送请求 [代理案例 1](https://segmentfault.com/q/1010000008196143) [代理案例 2](https://blog.csdn.net/luzhongk/article/details/78772286)
